# 更新日志

所有重要的版本更新都会记录在此文件中。

## [1.12.0] - 2026-04-26

### 新增
- **云同步功能** - 支持通过 WebDAV 协议进行云端配置同步，数据加密保障安全
- **同步管理界面** - 提供完整的云同步配置、状态查看和操作界面

### 优化
- **UI 体验改进** - 优化云同步相关界面交互和状态显示
- **稳定性提升** - 修复内存泄漏问题，增强错误处理

### 修复
- **代码清理** - 移除废弃组件，优化导航逻辑

## [1.11.2] - 2026-04-25

### 新增

- **统一表单验证** - 新增验证器工具，提供更友好的字段格式校验和错误提示
- **标准化错误处理** - 统一错误提示显示，提供更清晰的错误信息

### 优化

- **深色主题体验** - 完善禁用状态样式和暗色主题配色，视觉一致性更好
- **API 配置切换** - 切换配置时列表不再闪烁，体验更流畅

### 修复

- **空状态显示** - 修复技能和命令列表为空时的显示问题
- **表单验证** - 修复 API 配置表单必填验证逻辑，确保数据正确保存

## [1.10.1] - 2026-04-23

### 新增

- **后台下载功能** - 更新可在后台静默下载，不影响用户正常使用应用
- **下载进度实时显示** - 在设置页面的关于区域实时显示后台下载进度条和百分比
- **一键安装** - 后台下载完成后自动显示安装按钮，无需切换到更新弹窗即可完成更新
- **更新弹框后台选项** - 检测到更新时，可选择「后台下载」将下载任务移至后台执行

### 优化

- **更新流程更流畅** - 后台下载与前台下载的 UI 状态同步更准确
- **稳定性提升** - 完善更新相关测试用例，确保功能可靠

## [1.10.0] - 2026-04-22

### 新增

- **Commands 命令管理功能** - 新增命令管理模块，支持 iFlow 命令的查看、执行和管理界面

### 优化

- **自动更新功能** - 支持开关控制和静默检查模式，更新检查更灵活

## [1.9.0] - 2026-04-22

- **下载取消支持** - 可随时取消正在进行的更新下载

### 优化

### 修复

- **取消下载功能** - 修复取消下载按钮无法中断下载过程的问题

## [1.8.0] - 2026-04-19

## [1.5.1] - 2026-04-17

### 新增
- **系统托盘功能**
  - 窗口关闭时最小化到托盘而非退出应用
  - 托盘右键菜单：显示主窗口、切换 API 配置、退出
  - 双击托盘图标恢复显示主窗口
  - 从托盘快速切换 API 配置（自动保存当前配置）

### 优化
- API 配置编辑对话框的数据回填逻辑，使用 `(profile && profile.xxx) || fallback` 模式
- 改进 API 配置保存逻辑，确保切换配置时自动同步保存
- 移除 API 配置切换时的未保存更改确认弹窗

### 修复
- MCP 服务器保存及消息对话框被遮挡问题
- API 配置编辑功能，确保编辑按钮显示正确的配置数据
- 弹框点击空白处误关闭的问题
